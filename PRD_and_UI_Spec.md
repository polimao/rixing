# TODO 桌面端应用产品需求文档 (PRD) 兼 UI 设计规范

## 1. 产品概述

### 1.1 产品定位
一款基于 Electron 构建的极简、轻量级桌面待办事项（TODO）管理工具。主打无感操作、快速记录与高效查阅，通过常驻托盘和便捷的窗口交互，为用户提供沉浸式的工作/生活任务管理体验。支持 iCloud Drive 数据同步、番茄钟专注模式和智能分组管理。

### 1.2 核心价值
- **极简无压**：剔除冗余功能，只保留最核心的记录、分类、排序和状态管理。
- **所见即所得**：任务编辑与展示一体化，无需跳转页面，操作流畅。
- **视觉清新**：基于现代化 UI 设计，色彩柔和，布局紧凑，类似微信或现代 macOS 应用风格。
- **数据安全**：支持 iCloud Drive 自动同步，确保数据安全可靠。
- **专注高效**：内置番茄钟专注模式，提升工作效率。

## 2. 功能需求详情

### 2.1 窗口与系统级交互
- **托盘常驻 (Tray)**
  - 应用启动后常驻系统托盘，使用透明图标实现无干扰体验。
  - 托盘标题支持动态更新，实时显示当前未完成的待办数量（例如：`待办(3)`）。
  - 点击托盘图标，可在鼠标位置附近智能定位并呼出/隐藏主窗口。
- **窗口行为**
  - **失去焦点自动隐藏**：主窗口在失去焦点时自动隐藏，保持桌面整洁。
  - **窗口置顶 (Pin)**：提供"置顶"按钮。激活置顶后，窗口失去焦点也不会隐藏。
  - **番茄钟模式**：进入番茄钟专注模式时，窗口自动置顶并覆盖全屏应用，在所有工作区可见。
  - **无边框设计**：去除系统默认的标题栏和边框，使用自定义的圆角和阴影，顶部保留一个指向托盘的"小箭头"增加物理关联感。
  - **动态窗口高度**：窗口高度根据内容自动调整，支持平滑过渡动画。
- **全局快捷键**
  - `CmdOrCtrl+Shift+U`：快速切换窗口显示/隐藏状态。

### 2.2 数据存储与同步
- **iCloud Drive 同步**：
  - 数据自动存储在 iCloud Drive 的 `Todos` 文件夹中。
  - 支持从旧版本数据路径自动迁移到 iCloud Drive。
  - 避免覆盖用户真实数据的智能检测机制。
- **本地文件存储**：
  - 使用 JSON 格式存储任务数据和设置。
  - 自动创建必要的存储目录。

### 2.3 任务列表管理
- **任务结构**
  - 每条任务包含：唯一ID、任务内容、分类标签、优先级标签、状态标签、创建时间、完成时间。
  - 支持任务分组显示，根据排序规则智能分组。
- **智能分组系统**
  - **按分类+优先级分组**：相同分类和优先级的任务自动分组显示。
  - **按优先级分组**：仅按优先级分组显示。
  - **按时间分组**：不分组，按创建时间排序。
  - 每组任务提供"在此组添加"按钮，快速添加同分类同优先级任务。
- **新建任务**
  - 通过组内"添加"按钮或空状态时的默认按钮创建新任务。
  - 新任务自动继承所在组的分类和优先级设置。
  - 创建后自动滚动至该任务，输入框获得焦点，并伴有"抖动"动画。
- **任务编辑与展示**
  - **行内编辑**：直接在列表的任务项中进行文本输入，无需弹窗。
  - **自适应高度**：
    - 文本框默认单行显示，高度 `44px`。
    - 获得焦点时自动展开为多行（最高限制 `84px`，约三行高度）。
  - **智能文本截断**：
    - 使用二分查找算法动态截断超长文本。
    - 失去焦点时自动截取首行并显示省略号。
  - **快捷键支持**：
    - `Enter`：完成编辑，失去焦点并保存。
    - 空文本自动删除任务。
- **状态流转**
  - 任务状态分为："待办" 和 "完成"。
  - 切换为"完成"时触发滑出动画，并显示撤销操作的 Toast 提示。
  - 已完成任务显示完成时间。
- **任务操作**
  - **删除**：通过"更多操作"下拉菜单删除任务，支持撤销操作。
  - **专注模式**：为单个任务启动番茄钟专注模式。

### 2.4 排序与分组功能
- **排序规则**：
  - **默认排序**：按分类 → 优先级 → 时间排序。
  - **按优先级排序**：优先级 → 时间排序。
  - **按时间排序**：按创建时间倒序排列。
- **排序界面**：
  - 底部工具栏提供排序按钮和下拉菜单。
  - 当前排序规则高亮显示。

### 2.5 仪表盘与数据统计 (Dashboard)
- **数据卡片**：
  - 左侧显示"待办"和"已完成"任务数量。
  - "已完成"卡片可点击，弹出历史记录模态框。
- **趋势图表**：
  - 使用 Chart.js 绘制最近6个月的任务完成/新增趋势图。
  - 支持平滑曲线和数据标签显示。
  - 实时更新图表数据。

### 2.6 番茄钟专注模式
- **专注界面**：
  - 进入专注模式后窗口变为横向胶囊形状。
  - 显示当前专注任务标题和倒计时。
- **计时功能**：
  - 默认25分钟专注时间。
  - 支持暂停/继续功能。
  - 时间到自动提醒。
- **退出专注**：
  - 支持手动退出专注模式。
  - 自动恢复窗口原始尺寸和位置。

### 2.7 用户体验增强
- **空状态提示**：
  - 根据当前时间显示不同的问候语和图标。
  - 提供快速添加任务的入口。
- **Toast 提示**：
  - 任务完成、删除等操作提供撤销功能的 Toast 提示。
  - 4秒自动消失，支持手动关闭。
- **加载状态**：
  - 应用启动时显示加载动画。
  - 确保 DOM 完全渲染后隐藏加载界面。

## 3. UI/UX 设计规范

### 3.1 全局视觉
- **主色调**：
  - 品牌蓝（主交互）：`#409eff` (悬停/选中背景 `#ecf5ff`)
  - 成功绿（生活/低优）：`#67c23a` (背景 `#f0f9eb`)
  - 警告红（高优/删除）：`#db3e35`
  - 提示橙（中优）：`#f1b714`
  - 中性灰：`#909399`
- **字体与排版**：
  - 系统原生字体：`-apple-system, BlinkMacSystemFont, "PingFang SC"...`
  - 像素字体：`"Press Start 2P", "VT323", "Courier New", monospace`（用于数字显示）
  - 主文本大小：`14px`，辅助文本：`12px` 或 `11px`。
- **背景与边框**：
  - 主容器背景：`rgba(245, 247, 250, 0.4)` 带毛玻璃效果
  - 边框与分割线使用柔和的浅灰色 `#ebeef5` 或 `#dcdfe6`。
  - 阴影：`box-shadow: 0 8px 24px rgba(0, 0, 0, 0.08)` 营造悬浮感。

### 3.2 组件设计规范
- **任务组 (Todo Group)**：
  - 背景：`#eee`，圆角 `4px`，轻微阴影
  - 组头：显示分类和优先级标签，右侧添加按钮
- **任务项 (Todo Item)**：
  - 高度：`44px`，支持动态调整
  - 悬停效果：轻微背景变灰 `rgba(0, 0, 0, 0.02)`
  - 输入框：聚焦时显示白色背景和蓝色边框
- **标签样式 (Badges)**：
  - 分类标签：工作（蓝色），生活（绿色）
  - 优先级标签：高（红色），中（橙色），低（绿色）
  - 状态标签：待办（灰色边框），完成（灰色背景）
- **模态框 (Modal)**：
  - 已完成列表模态框：白色背景，圆角 `12px`
  - 番茄钟模态框：透明背景，专注界面

### 3.3 交互细节 (Micro-interactions)
- **Hover 反馈**：
  - 按钮：悬浮时背景变深，显示 Tooltip
  - 任务项：悬浮时浮现操作按钮
- **动画效果**：
  - **抖动动画**：新建/复制任务时 `600ms` 抖动效果
  - **滑出动画**：完成任务时向右滑出
  - **高度过渡**：文本框展开/收起平滑过渡
  - **窗口动画**：高度调整时的平滑过渡
- **无障碍设计**：
  - 移除所有原生控件的默认焦点边框
  - 自定义焦点状态样式
  - 隐藏 Webkit 滚动条，保持界面极简

### 3.4 暗色模式支持
- **自动适配**：通过 `@media (prefers-color-scheme: dark)` 自动切换
- **颜色映射**：
  - 浅色背景 → 深色背景 `rgba(30, 32, 34, 0.7)`
  - 文字颜色适配深色主题
  - 保持相同的视觉层次和对比度

## 4. 技术实现备忘

### 4.1 核心架构
- **前端框架**：原生 HTML/CSS/JavaScript + Electron
- **图表库**：Chart.js 用于数据可视化
- **数据存储**：JSON 文件 + iCloud Drive 同步

### 4.2 关键技术实现
1. **动态窗口高度**：通过计算内容高度 + 平滑动画实现自适应窗口
2. **智能文本截断**：二分查找算法 + scrollWidth/clientWidth 比较
3. **番茄钟模式**：窗口状态管理 + 跨工作区显示
4. **数据迁移**：旧版本数据自动迁移到 iCloud Drive
5. **分组系统**：基于排序规则的动态分组算法

### 4.3 性能优化
- **虚拟滚动**：长列表性能优化（如有需要）
- **防抖处理**：频繁操作的事件防抖
- **内存管理**：定时器清理和事件监听器管理

### 4.4 兼容性考虑
- **macOS 优化**：Dock 隐藏、原生动画支持
- **跨平台支持**：Windows/Linux 基础功能兼容
- **系统集成**：托盘图标、全局快捷键、文件系统访问
